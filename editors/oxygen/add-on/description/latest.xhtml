This add-on installs a recent snapshot of the development version of XSpec. See Wiki for details.

This add-on does not work side by side with XSpec Framework add-on (a companion add-on for XSpec Helper view add-on).

Changelog (not inclusive)

v3.3.2

  • Official release of v3.3. For features, see v3.3.1 listing below.
  • Tested with Saxon 12.9, 11.7, and 10.9.
  • Note: In your Oxygen settings, please configure Options > Preferences > Add-ons with the update site URL https://xspec.io/editors/oxygen/oxygen-addon.xml instead of https://github.com/xspec/xspec/raw/master/oxygen-addon.xml because the latter URL will not continue to be updated.

v3.3.1

  • Release Candidate of the stable release
  • You can execute an XSpec test for XSLT or XQuery using an XProc 3 pipeline (#2127)
  • You can execute an XSpec test for a Schematron schema that has an XQuery-based queryBinding, using XQS as the Schematron implementation (#2157)
  • In the x:expect element, the new, optional 'result-type' attribute describes the sequence type you expect for the actual result. XSpec checks the type before evaluating @test, so an unexpected type leads to an ordinary failure instead of a type error. (#2122)
  • Tested with BaseX 12.0

v3.2.2

  • Official release of v3.2

v3.2.1

  • Release Candidate of the stable release
  • Code coverage report includes Contents section with coverage statistics (#2078).
  • Tested with BaseX 11.7.

v3.2.0

  • Test reports use black text on white background by default, improving accessibility. Themes 'whiteblack' (white on black) and 'classic' (earlier green/pink design) are also available. (#2055)
  • Tests for Schematron can verify string values of Schematron properties (#2045).
  • Tested with BaseX 11.6
  • Tested with Oxygen 27.0.

v3.1.3

  • Fix for xspec/xspec-maven-plugin-1 and other uses of XSpec from a .zip or .jar file
  • SchXslt 1.10.1 replaces 1.10 as the built-in Schematron implementation

v3.1.2

  • Official release of v3.1

v3.1.1

  • fix(report): XSLT code coverage reports for Saxon 12.4 have many bug fixes in the coverage status
  • feat(schematron): SchXslt 1.10 replaces 1.9.5 as the built-in Schematron implementation
  • Tested with Saxon 10.9, 11.6, and 12.5, except that detailed contents of XSLT code coverage reports are tested with Saxon 12.4
